# 🛒 Sanjay Kirana Store

A full-stack grocery management and online ordering system for our family-owned kirana store in Harraiya, Basti.

## 🚀 Tech Stack

- Frontend: React
- Backend: Flask
- Database: PostgreSQL

## 📌 Features (Planned)

- Product Search
- Categories
- Shopping Cart
- Order Placement
- Admin Dashboard
- Inventory Management
- Customer Management
- Sales Analytics
